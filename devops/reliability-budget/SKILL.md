# Reliability Budget

Use this skill to manage SLOs and release risk.

## Workflow

1) Define SLOs and error budget windows.
2) Track consumption and alert thresholds.
3) Gate releases when budget is exhausted.
4) Report budget status.

## Output expectations

- Provide SLOs, current budget, and gating decisions.
