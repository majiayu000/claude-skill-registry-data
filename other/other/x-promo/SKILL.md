# X投稿作成スキル（ハーネス専用・非配布）

> **注意**: このスキルは claude-code-harness 開発専用です。`.gitignore` で除外されており、プラグイン配布には含まれません。

## 概要

Claude harness のプロモーション用X投稿を作成するスキルです。
Xアルゴリズム最適化のノウハウを活用して、効果的な投稿を生成します。

---

## 3つの基本原則

### ① プロフィールが超重要

知名度の高い単語を複数入れる。投稿にも同じ単語を含める。

**プロフィールキーワード例**:
- Google, AI, ChatGPT, Claude Code
- Vibecoder, ノーコード, AI開発
- エンジニア, 個人開発

### ② Xアルゴリズムに「有益」と思わせる

判断ポイント:
- 投稿の **独自性** と **わかりやすさ**
- 既存フォロワーに **評価される内容** か

### ③ プロフィールキーワードを投稿に含める

プロフィールに書いていないカテゴリは扱わない。1アカウントで扱えるトピックは限られている。

---

## 投稿作成ガイドライン

### バズと既存フォロワー向けの比率

| タイプ | 割合 | 目的 |
|--------|------|------|
| 既存フォロワー向け | 80% | 固定ファンの維持 |
| バズ狙い | 20% | 新規リーチ拡大 |

### 強い行動

#### 1. 引用リポスト（超強行動）

```
構成:
1行目: 引用元に対して「明確な賛成 or 反対」を表明
2行目以降: 理由の補足、欠けている視点の追加

注意:
- 必ず画像または動画を添付（引用先の画像は消えやすい）
- 引用の引用の場合 → スクショを貼る
```

#### 2. インプレゾンビ（リプ欄攻略）

狙い目:
- 100万インプ以上
- コメント欄が一言感想ばかり
- 薄い内容のリプライが多い

戦略:
- エビデンスベースで論理的な指摘
- リプ欄の一番上に表示されやすくなる

### 投稿の品質管理

| ルール | 理由 |
|--------|------|
| 10分以内に1いいねなければ削除 | Xの評価を落とさない |
| URLは普通に貼る | フォロワーに不便を強いない |
| 一次情報・参考情報を示す | 信頼性向上 |

## バズの連鎖を意識する

1. 最初のバズは運（でも狙える）
2. バズったら、角度・視点を変えて派生投稿
3. 毎日どれかが10万インプを超える状態を目指す
4. 連日20万インプ↑で安定

### 派生の例

```
GWS関連がバズった
  ↓
Google → アメリカ
Gemini → 他のLLM
GWS契約 → お得なAIサービス紹介
```

---

## スペシャリストになれ

1. 最初は誰か1人の安定したいいねを目指す
2. 固定ファンを増やす
3. フォロワー100人超えたら伸びる可能性十分
4. 「バズってる型」を参考に言い回しを試行錯誤
5. バズりやすい単語を特定 → プロフィール更新

## 使い方

1. このスキルを起動
2. ターゲットを指定（エンジニア/バイブコーダー/興味ある人/再チャレンジ）
3. 伝えたい内容を伝える
4. 投稿案を生成
5. 必要に応じて調整
