# Supabase Tools Setup

Run `/supabase-tools:setup` to configure Supabase MCP.

## Quick Fixes

- **OAuth failed** - Re-authenticate via Supabase dashboard
- **Project not found** - Verify project_ref in config
- **Permission denied** - Check RLS policies

## Don't Need Supabase?

Disable via `/mcp` command to prevent errors.
