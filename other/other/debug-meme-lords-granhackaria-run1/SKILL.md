# GSD Debug Workflow

Workflow for systematic issue investigation and resolution.

## When to Use

- Bugs encountered
- Verification failures
- Unexpected behavior

## Phases

1. Describe problem
2. Investigate root cause
3. Implement fix
4. Verify solution
5. Prevent recurrence

## Entry Point

Run: `gsd:debug`

## Success Criteria

Issue resolved with verified fix.
