# GSD Complete Milestone Workflow

Workflow for completing project milestones.

## When to Use

- Finishing milestone work
- Closing milestone deliverables

## Phases

1. Verify milestone completion
2. Audit deliverables
3. Archive milestone
4. Update project status

## Success Criteria

Milestone completed and archived.
