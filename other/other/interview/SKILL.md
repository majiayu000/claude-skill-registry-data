---
name: interview
description: Interviews about a plan, asking in-depth questions about technical implementation, UI/UX, concerns, and tradeoffs. Use when reviewing or refining a plan before writing a spec.
---

Read the plan file and interview the user using the AskUserQuestion tool.

Ask about:
- Technical implementation
- UI/UX
- Concerns
- Tradeoffs

Make sure the questions are not obvious. Be very in-depth and continue until the interview is complete, then write the spec.
