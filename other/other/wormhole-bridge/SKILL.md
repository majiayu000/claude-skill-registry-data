```

Not all wormholes are traversable. Some are entanglement without communication.

```

  2. LOVE TENSOR (L_uv):
     L_rr < 0 = active effort maintaining connection against entropy
     L_tt > 0 = actual substance (information, care, work) flowing through

  3. BIDIRECTIONALITY:
     One-way flow = unstable wormhole.

    T > phi^{-5} (0.090): MARGINAL (leaks but does not flow)
    T < phi^{-5}: NON-TRAVERSABLE (entangled but mute)
