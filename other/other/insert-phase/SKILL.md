# GSD Insert Phase

Inserts a phase into the roadmap at specific position.

## When to Use

- Adding phase at specific position
- Reordering roadmap

## Process

1. Identify insertion point
2. Create phase
3. Update roadmap

## Success Criteria

Phase inserted into roadmap.
