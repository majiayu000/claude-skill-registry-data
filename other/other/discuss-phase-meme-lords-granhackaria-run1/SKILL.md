# GSD Discuss Phase Workflow

Workflow for discussing phase details with user.

## When to Use

- Clarifying phase scope
- Getting user feedback
- Making decisions

## Phases

1. Load phase details
2. Present information
3. Discuss with user
4. Document decisions

## Success Criteria

Discussion complete with decisions captured.
