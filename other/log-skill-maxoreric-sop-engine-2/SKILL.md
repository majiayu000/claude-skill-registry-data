# 执行记录

保存执行过程、结果和经验。

## 记录内容

### 执行日志
- 时间戳
- 执行的操作
- 输入和输出
- 结果状态

### 决策记录
- 做了什么决策
- 为什么这样决定
- 考虑过的其他选项

### 经验教训
- 什么做得好
- 什么可以改进
- 下次要注意什么

## 输出格式

### 工作日志 worklog.md

```markdown
# 工作日志

## [日期]

### [时间] - [操作]
- 输入：[描述]
- 输出：[描述]
- 状态：成功 / 失败
- 备注：[说明]

### [时间] - [操作]
...
```

### 决策记录 decisions.md

```markdown
# 决策记录

## [日期] - [决策主题]

### 背景
[为什么需要这个决策]

### 选项
1. 选项 A：[描述]
   - 优点：
   - 缺点：
2. 选项 B：[描述]
   - 优点：
   - 缺点：

### 决策
选择：[选项 X]
原因：[为什么]

### 后续
[需要什么后续行动]
```

## 存储位置

- Agent 相关：`agents/<agent-name>/logs/`
- Skill 相关：`.sop-engine/skills/<skill-name>/workspace/logs/`

## 原则

- 及时记录，不要事后回忆
- 记录事实，不加主观判断
- 保持格式一致
- 定期整理归档
