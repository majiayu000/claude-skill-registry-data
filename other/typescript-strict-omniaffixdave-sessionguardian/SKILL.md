# TypeScript Strict

Use strict TypeScript patterns:
- Use explicit types (avoid 'any')
- Use 'unknown' for external data and validate
- Handle null/undefined properly with optional chaining
- Use type guards and type narrowing
- Prefer interfaces for object shapes
