# Marketing Audit - 総合監査スキル

マーケティング面の包括的な監査・分析を行うメタスキル。
各専門スキルを統合的に活用し、体系的な改善提案を行う。

## 監査対象領域

### 1. ページ・コンバージョン最適化

**使用スキル:**
- `lp-optimizer` - LP/HP/価格ページのCRO分析
- `signup-flow-cro` - 登録フロー最適化
- `form-cro` - フォーム最適化
- `onboarding-cro` - オンボーディング最適化
- `paywall-upgrade-cro` - アップグレード促進

**監査項目:**
- [ ] ヒーローセクションの効果
- [ ] CTAの明確さと配置
- [ ] 社会的証明の活用
- [ ] 登録フローの摩擦点
- [ ] オンボーディング完了率
- [ ] 無料→有料転換率

### 2. SEO・検索対策

**使用スキル:**
- `seo-audit` - 技術SEO・オンページSEO

**監査項目:**
- [ ] 技術SEO（クロール可能性、インデックス状況）
- [ ] オンページSEO（タイトル、メタ、見出し）
- [ ] 構造化データ（JSON-LD）
- [ ] Core Web Vitals
- [ ] モバイルフレンドリー

### 3. コンテンツ・コピー

**使用スキル:**
- `copywriting` - マーケティングコピー作成
- `email-sequence` - メールシーケンス設計

**監査項目:**
- [ ] バリュープロポジションの明確さ
- [ ] ベネフィット vs 機能のバランス
- [ ] ターゲット言語の使用
- [ ] ウェルカムメールの効果
- [ ] リテンションメールの設計

### 4. 分析・計測

**使用スキル:**
- `analytics-tracking` - GA4/GTM設定
- `ab-test-setup` - A/Bテスト設計

**監査項目:**
- [ ] コンバージョントラッキング設定
- [ ] ファネル計測
- [ ] UTMパラメータ運用
- [ ] A/Bテスト計画

### 5. 成長戦略

**使用スキル:**
- `pricing-strategy` - 価格設計
- `launch-strategy` - ローンチ戦略
- `referral-program` - リファラルプログラム

**監査項目:**
- [ ] 価格ティアの最適性
- [ ] ローンチチャネルの選定
- [ ] リファラルインセンティブ設計

### 6. 心理学・UX

**使用スキル:**
- `marketing-psychology` - マーケティング心理学
- `ux-psychology` - UX心理学

**監査項目:**
- [ ] 心理学原則の適用状況
- [ ] ダークパターンの回避
- [ ] 認知負荷の最小化

## 監査プロセス

### Phase 1: 現状把握

1. **サイト全体の確認**
   - トップページ → 価格ページ → 登録 → オンボーディング

2. **データ収集**（可能であれば）
   - 現在のコンバージョン率
   - 主要な離脱ポイント
   - 顧客フィードバック

### Phase 2: 領域別分析

各領域のスキルを参照し、チェックリストに沿って分析：

```
1. ページ最適化 → lp-optimizer, signup-flow-cro, ...
2. SEO → seo-audit
3. コンテンツ → copywriting, email-sequence
4. 分析 → analytics-tracking, ab-test-setup
5. 戦略 → pricing-strategy, launch-strategy, referral-program
6. 心理学 → marketing-psychology, ux-psychology
```

### Phase 3: 優先順位付け

**Impact × Effort マトリクス:**

| | 低労力 | 高労力 |
|---|---|---|
| **高インパクト** | 🔴 今すぐ実行 | 🟡 計画的に実行 |
| **低インパクト** | 🟢 余裕があれば | ⚪ 後回し |

### Phase 4: 実行計画

**推奨フォーマット:**

```markdown
## 改善提案: [タイトル]

**関連スキル:** lp-optimizer, copywriting
**優先度:** 🔴 Critical / 🟡 Important / 🟢 Nice-to-have
**インパクト:** コンバージョン率 +X% 見込み

### 現状
[問題の説明]

### 提案
[具体的な改善案]

### 実装手順
1. ...
2. ...

### 成功指標
- [KPI]
```

## クイックスタート

「マーケティング面を全面的に調査検討して」と言われたら：

1. **ユーザーに確認:**
   - 対象URL/プロダクト
   - 特に気になっている領域
   - 現在の主要KPI

2. **領域別にスキルを参照して分析:**
   - 最も懸念される領域から開始
   - 各スキルのチェックリストを活用

3. **統合レポート作成:**
   - 領域別の発見事項
   - 優先度付き改善提案
   - 実行ロードマップ
