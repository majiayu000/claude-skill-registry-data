# GSD Discuss Phase

Discusses phase details with user.

## When to Use

- Clarifying phase scope
- Discussing approach
- Getting user feedback

## Process

1. Load phase details
2. Present information
3. Discuss with user
4. Capture decisions

## Success Criteria

Discussion completed with decisions captured.
