# UI Libraries & Components

## Instructions

1. **Library selection**

   - Shadcn UI, Universal UI, or similar
   - Lightweight and maintainable

2. **Reusable components**

   - Buttons, modals, task lists
   - Input fields with validation

3. **Integration**
   - Align library styling with project theme
   - Ensure compatibility with Next.js App Router

## Best Practices

- Keep components modular and importable
- Avoid conflicts with existing styles
- Prioritize accessibility and responsiveness
