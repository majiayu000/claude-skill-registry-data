# Lorepedia Notion

Lorepediaプロジェクトの情報はすべてNotionに記載されている。必要な情報を検索・取得する。

## 自動トリガー条件

以下の状況で自動的にNotionを検索する：

- 機能の実装方法が不明な時
- DB構造やテーブル設計が必要な時
- 技術スタックやライブラリの選定理由を確認したい時
- 画面設計やユーザーフローを確認したい時
- プロジェクトの要件や仕様を確認したい時

## 検索手順

1. `notion-search` で関連ページを検索
2. 見つかったページを `notion-fetch` で内容取得
3. 必要に応じて追加キーワードで再検索

## 検索キーワード例

| 情報種別 | 検索キーワード |
|---------|--------------|
| 技術選定 | 技術選定, スタック, アーキテクチャ |
| DB設計 | テーブル, スキーマ, データベース, DB |
| 機能要件 | 機能, 要件, ユースケース |
| 画面設計 | 画面, UI, デザイン, ワイヤーフレーム |
| API | API, エンドポイント, インターフェース |

## Notion MCPツール

- `notion-search`: キーワードでページを検索
- `notion-fetch`: ページIDまたはURLから内容を取得

## 使用例

```
# 技術スタックを確認
notion-search(query="技術選定 スタック")

# 検索結果のページ内容を取得
notion-fetch(id="<page-id>")

# DB設計を確認
notion-search(query="テーブル設計 スキーマ")
```
