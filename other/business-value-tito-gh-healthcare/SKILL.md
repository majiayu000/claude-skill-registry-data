# Business Value Skill

## 目的
研究案のビジネス価値を体系的に評価し、商業的機会、競合優位性、パートナーシップ可能性を定量化する。

## 入力パラメータ

| パラメータ | 必須 | 説明 | 例 |
|-----------|------|------|-----|
| RESEARCH_PROPOSAL | Yes | 研究案の概要 | DRd vs VRd比較効果研究 |
| DRUG | Yes | 対象薬剤 | daratumumab |
| INDICATION | Yes | 対象疾患 | NDMM |
| SPONSOR_TYPE | No | 想定スポンサー | Pharma/Academia/Regulator |
| MARKET_CONTEXT | No | 市場環境 | Competitive, growing |
| ESTIMATED_COST | No | 推定研究費用 | ¥50M |
| TIMELINE | No | 想定期間 | 18 months |

## ビジネス価値評価フレームワーク

### 1. 収益機会（Revenue Opportunity）

| 評価項目 | 評価基準 | スコア (1-5) |
|----------|----------|-------------|
| **製薬企業ニーズ合致** | スポンサー獲得可能性 | |
| **市場規模** | 対象薬剤・疾患の市場 | |
| **契約規模ポテンシャル** | 研究委託費・ライセンス | |
| **追加研究展開可能性** | 後続研究・長期契約 | |
| **複数顧客可能性** | 競合製薬企業等への展開 | |

### 2. 競合優位性（Competitive Advantage）

| 評価項目 | 評価基準 | スコア (1-5) |
|----------|----------|-------------|
| **データ独自性** | 他社DBで実施不可能な程度 | |
| **EHR-RWD優位性** | クレームDB（JMDC/MDV）との差別化 | |
| **方法論的優位性** | 解析スキル・ノウハウ | |
| **スピード優位性** | 競合より早く実施可能か | |
| **品質優位性** | アウトプット品質での差別化 | |

### 3. パートナーシップ機会（Partnership Opportunity）

| 評価項目 | 評価基準 | スコア (1-5) |
|----------|----------|-------------|
| **製薬企業連携** | 共同研究・委託研究可能性 | |
| **アカデミア連携** | 学術機関との共著・共同研究 | |
| **規制当局連携** | PMDA相談・パイロット可能性 | |
| **学会連携** | 学会発表・ガイドライン貢献 | |
| **国際連携** | 海外組織との連携可能性 | |

### 4. 知財・パブリケーション価値（IP & Publication Value）

| 評価項目 | 評価基準 | スコア (1-5) |
|----------|----------|-------------|
| **高IF誌掲載可能性** | 一流誌掲載の見込み | |
| **引用インパクト** | 広く引用される可能性 | |
| **ノウハウ蓄積** | 社内知見・ケイパビリティ向上 | |
| **方法論確立** | 汎用可能な手法開発 | |
| **ケーススタディ価値** | 営業・PR活用可能性 | |

### 5. ブランド・レピュテーション価値（Brand Value）

| 評価項目 | 評価基準 | スコア (1-5) |
|----------|----------|-------------|
| **業界認知向上** | RWD領域でのプレゼンス | |
| **信頼性向上** | エビデンス品質での評価 | |
| **イノベーター認知** | 革新的取り組みとしての評価 | |
| **CSR/社会貢献** | 患者・社会への貢献認知 | |
| **採用・人材獲得** | 優秀人材へのアピール | |

## ビジネス価値スコアリング

### 総合ビジネス価値スコア算出

```
Business Value Score =
  (Revenue Opportunity × 0.30) +
  (Competitive Advantage × 0.25) +
  (Partnership Opportunity × 0.20) +
  (IP & Publication × 0.15) +
  (Brand Value × 0.10)
```

### ROI推定

```
Estimated ROI = (Expected Revenue - Estimated Cost) / Estimated Cost × 100%

Expected Revenue = Σ(Revenue Source × Probability × Timeline Factor)
```

## 注意事項

1. **収益予測の保守性**: 楽観的すぎる収益予測を避ける
2. **競合分析の客観性**: 自社優位性の過大評価を避ける
3. **差別化の具体性**: JMDC/MDVとの差別化を具体的・定量的に示す
4. **リスク認識**: ビジネスリスクを適切に評価・開示
5. **機密保持**: 競合情報・価格情報の取り扱いに注意
