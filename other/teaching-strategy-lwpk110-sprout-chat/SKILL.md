# 教学策略技能

## 问题类型分类

| 类型 | 标识 | 关键词 | 难度 |
|------|------|--------|------|
| 加法 | ADDITION | 加、一共有、总共 | ⭐ |
| 减法 | SUBTRACTION | 减、剩下、吃掉 | ⭐ |
| 乘法 | MULTIPLICATION | 乘、倍、每组 | ⭐⭐ |
| 除法 | DIVISION | 除、平均、分给 | ⭐⭐ |
| 比较 | COMPARISON | 大、小、多、少 | ⭐⭐ |
| 应用题 | WORD_PROBLEM | 有、个、还、最后 | ⭐⭐⭐ |

---

## 教学策略模板

### 加法策略
**默认比喻**: 积木
```python
ADDITION_STRATEGY = {
    "template": "我们来玩堆积木的游戏吧！想象一下...",
    "questions": [
        "这里有几个积木？",
        "我们再加几个积木？",
        "现在一共有几个了？"
    ]
}
```

### 减法策略
**默认比喻**: 苹果
```python
SUBTRACTION_STRATEGY = {
    "template": "我们来分苹果吧！本来有...",
    "questions": [
        "原来有几个苹果？",
        "拿走了几个？",
        "现在还剩几个？"
    ]
}
```

## 策略选择流程

```
问题输入 → 提取关键词 → 匹配问题类型 → 选择教学策略 → 生成引导问题
```

## 比喻选择规则

| 年龄段 | 推荐比喻 | 避免比喻 |
|--------|----------|----------|
| 6岁 | 糖果、积木、小动物 | 电子产品、抽象概念 |
| 7岁 | 玩具、水果、动物 | 过于幼稚的比喻 |
