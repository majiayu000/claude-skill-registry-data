## Brand Strategy Framework

### Brand Pyramid
```
           Purpose
             ↑
         Personality
             ↑
         Benefits
             ↑
        Attributes
             ↑
      Product/Service
```

### Positioning Statement
```
Для [target customer]
Кто [need/opportunity]
[Brand] — это [category]
Который [key benefit]
В отличие от [competitors]
Мы [differentiator]
```

## Brand Voice Dimensions

| Dimension | Spectrum |
|-----------|----------|
| Formal ↔ Casual | Насколько профессионально |
| Serious ↔ Playful | Сколько юмора |
| Expert ↔ Accessible | Насколько технично |
| Reserved ↔ Enthusiastic | Насколько энергично |

## Brand Voice Guidelines

### Примеры для разных тонов

**Formal + Expert:**
```
"Наша платформа обеспечивает комплексную автоматизацию
маркетинговых процессов с использованием передовых
алгоритмов машинного обучения."
```

**Casual + Accessible:**
```
"Забудь о рутине! Наш AI сделает всю работу за тебя —
просто загрузи креативы и смотри, как растут продажи."
```

## Brand Messaging Framework

### Message House
```
                    [Brand Promise]
                          |
    [Pillar 1]      [Pillar 2]      [Pillar 3]
         |               |               |
    [Proof]         [Proof]         [Proof]
    [Proof]         [Proof]         [Proof]
                          |
              [Reason to Believe]
```

### Elevator Pitch Template
```
[Company] помогает [target audience]
[solve problem/achieve goal]
через [unique approach/solution].
В отличие от [alternatives],
мы [key differentiator]
что приводит к [measurable outcome].
```

## Brand Architecture

### Types
- **Branded House**: Один мастер-бренд (Google)
- **House of Brands**: Независимые бренды (P&G)
- **Endorsed Brands**: Sub-brands с endorsement (Marriott Bonvoy)
- **Hybrid**: Комбинация подходов

## Rebranding Triggers

1. **Business changes**
   - Mergers & acquisitions
   - New markets
   - Product pivots

2. **Market changes**
   - Competitor evolution
   - Customer expectations shift
   - Industry disruption

3. **Brand issues**
   - Outdated perception
   - Negative associations
   - Inconsistent identity

## Лучшие практики

1. **Consistency is key** — единообразие во всех touchpoints
2. **Document everything** — поддерживайте актуальные guidelines
3. **Train the team** — все должны понимать бренд
4. **Monitor & measure** — регулярно оценивайте brand health
5. **Evolve thoughtfully** — обновляйте, но сохраняйте core identity
