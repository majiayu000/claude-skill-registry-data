# Family Travel

## Overview

This skill provides guidance on planning trips with children, including age-appropriate activity selection, family-friendly accommodations, and practical packing considerations.

## Usage

The hotel-researcher and activity-curator sub-agents reference this skill when:
- Filtering accommodations for family-friendliness
- Selecting activities appropriate for children's ages
- Adjusting itinerary pacing for families
- Recommending family-friendly restaurants

## Age-Based Considerations

### Young Children (0-5)
- Nap time: Build in rest periods
- Stroller access: Check venue accessibility
- Simple meals: Familiar food options available
- Short attention spans: Quick activities, variety

### School Age (6-12)
- Interactive experiences: Hands-on museums, activities
- Some independence: Can handle moderate walking
- Learning opportunities: Historical sites with stories
- Energy: Need physical outlets

### Teens (13+)
- Include in planning: Let them choose some activities
- Technology: WiFi availability matters
- Later nights: Can handle evening activities
- Independence: Some solo exploration okay
