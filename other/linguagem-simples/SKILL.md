Guia para escrever textos acessíveis para o público do Tá na Mão (5ª série de escolaridade).

## Princípios

1. **Frases curtas** - Máximo 15-20 palavras
2. **Palavras comuns** - Evitar jargões
3. **Voz ativa** - "Você pode" em vez de "É possível"
4. **Ações claras** - Dizer exatamente o que fazer

## Transformações

### Termos Técnicos → Simples

| Técnico | Simples |
|---------|---------|
| Cadastro Único | Cadastro do governo |
| Beneficiário | Pessoa que recebe |
| Elegível | Tem direito |
| Protocolar | Entregar papéis |
| Deferido | Aprovado |
| Indeferido | Não aprovado |
| Vigência | Tempo que vale |
| Renda per capita | Dinheiro por pessoa da família |
| NIS | Número do Cadastro Único |
| CPF | Número do CPF |

### Frases Longas → Curtas

❌ "Para a realização da consulta de elegibilidade aos programas sociais, é necessário que o cidadão apresente a documentação comprobatória no posto de atendimento mais próximo de sua residência."

✅ "Para saber se você tem direito aos benefícios, leve seus documentos ao CRAS mais perto de você."

### Voz Passiva → Ativa

❌ "O benefício será depositado na conta indicada."
✅ "O governo vai depositar o dinheiro na sua conta."

❌ "A documentação deve ser apresentada."
✅ "Você precisa levar os documentos."

## Templates

### Mensagem de Benefício Encontrado
```
Encontrei {programa} no seu nome!

Valor: R$ {valor} por mês
Você recebe desde: {data}

Quer saber mais sobre esse benefício?
```

### Mensagem de Requisitos
```
Para ter direito ao {programa}, você precisa:

1. {requisito1}
2. {requisito2}
3. {requisito3}

Você se encaixa nesses requisitos?
```

### Mensagem de Documentos
```
Leve esses documentos ao CRAS:

- RG ou carteira de identidade
- CPF
- Comprovante de onde você mora
- Comprovante de quanto você ganha

Precisa de ajuda para conseguir algum documento?
```

### Mensagem de Erro
```
Não consegui encontrar essa informação agora.

Você pode:
1. Tentar de novo em alguns minutos
2. Ir ao CRAS mais perto
3. Ligar para o 121

Quer que eu procure o CRAS mais perto de você?
```

## Checklist de Revisão

- [ ] Frases têm no máximo 20 palavras?
- [ ] Usei palavras do dia a dia?
- [ ] As ações estão claras?
- [ ] Uma criança de 10 anos entenderia?
- [ ] Evitei siglas sem explicação?
- [ ] Usei "você" em vez de "o cidadão"?

## Teste de Legibilidade

Leia o texto em voz alta:
- Se precisar respirar no meio da frase, ela está longa demais
- Se tropeçar em alguma palavra, troque por uma mais simples
- Se não souber explicar para uma criança, simplifique
