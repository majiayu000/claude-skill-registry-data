Metodologia para pesquisar novos benefícios, legislação e fontes de dados.

## Processo de Pesquisa

### 1. DEFINIR: Escopo da Pesquisa
```
Pergunta central: [O que preciso descobrir?]
Fontes prováveis: [Onde encontrar?]
Critério de sucesso: [Como sei que terminei?]
```

### 2. COLETAR: Buscar Informações

#### Fontes Oficiais
- **gov.br** - Portal do governo federal
- **MDS** - Ministério do Desenvolvimento Social
- **INSS** - Previdência e BPC
- **Caixa** - Operadora de benefícios
- **IBGE** - Dados demográficos

#### Legislação
- **planalto.gov.br** - Leis federais
- **lexml.gov.br** - Busca de legislação
- **Diário Oficial** - Portarias e decretos

#### Dados Abertos
- **dados.gov.br** - Portal brasileiro de dados abertos
- **basedosdados.org** - Datasets tratados
- **SAGI/MDS** - Relatórios sociais

### 3. ANALISAR: Processar Informações

```markdown
## Achado 1: [Título]
- Fonte: [URL ou referência]
- Data: [Quando publicado]
- Relevância: [Por que importa]
- Confiabilidade: [Alta/Média/Baixa]

## Achado 2: [Título]
...
```

### 4. SINTETIZAR: Consolidar Descobertas

```markdown
## Resumo Executivo
[2-3 parágrafos com principais descobertas]

## Detalhes
[Informações completas organizadas]

## Próximos Passos
[O que fazer com essa informação]

## Fontes
[Lista de todas as fontes consultadas]
```

## Ferramentas de Pesquisa

### Web Search
```
Usar WebSearch para encontrar informações atuais
Query exemplo: "bolsa família requisitos 2024 site:gov.br"
```

### Web Fetch
```
Usar WebFetch para extrair conteúdo de páginas específicas
URL: https://www.gov.br/mds/...
```

## Checklist de Qualidade

- [ ] Informação vem de fonte oficial?

- [ ] Legislação citada está vigente?
