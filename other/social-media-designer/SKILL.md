# Social Media Designer

A social media design expert that creates platform-optimized graphics for maximum engagement. This skill combines platform best practices, visual design principles, and content strategy to produce scroll-stopping social media content that drives results.

From Instagram posts to Twitter headers to LinkedIn carousels, this skill ensures your social content looks professional, on-brand, and optimized for each platform's unique requirements.

## Core Workflows

### Workflow 1: Instagram Content Suite
1. **Define content type**
   - Feed post (single or carousel)
   - Story
   - Reel cover
   - Highlight cover
   - Profile picture

2. **Design feed post**
   - Dimensions: 1080x1080px (square) or 1080x1350px (portrait)
   - Visual hierarchy for quick scanning
   - Brand colors and fonts
   - On-brand photography or graphics
   - Text overlay (readable on mobile)

3. **Create story template**
   - Dimensions: 1080x1920px
   - Safe zones (avoid top/bottom 250px)
   - Interactive elements (polls, questions, sliders)
   - Swipe-up or link sticker area
   - Consistent brand frame

4. **Build content series**
   - Template for consistency
   - Color-coded by category
   - Reusable elements
   - Variation within system

### Workflow 2: LinkedIn Professional Graphics
1. **Choose format**
   - Single image post: 1200x627px
   - Carousel: 1080x1080px per slide
   - Article header: 1200x627px
   - Company page cover: 1128x191px
   - Profile banner: 1584x396px

2. **Design for platform**
   - Professional, polished aesthetic
   - Data-driven content
   - Industry-relevant imagery
   - Thought leadership focus
   - Clear value proposition

3. **Create carousel**
   - Compelling cover slide
   - One idea per slide
   - Consistent template
   - Strong conclusion/CTA
   - Page numbers or progress indicator

4. **Optimize for feed**
   - Hook in first 2 lines
   - Visual supports text
   - Professional photography
   - Avoid overly promotional

### Workflow 3: Twitter/X Visual Content
1. **Design tweet images**
   - Dimensions: 1200x675px (16:9)
   - High contrast for timeline
   - Bold, simple message
   - Works as thumbnail
   - Text readable at small size

2. **Create header image**
   - Dimensions: 1500x500px
   - Brand expression
   - Seasonal or campaign updates
   - Visual interest, not busy
   - Works with profile picture

3. **Design thread graphics**
   - Consistent style across thread
   - Numbered or sequential
   - Eye-catching colors
   - Quote graphics
   - Data visualizations

### Workflow 4: Multi-Platform Campaign
1. **Design master concept**
   - Core message/visual
   - Adaptable composition
   - Flexible color system
   - Reusable elements

2. **Adapt for each platform**
   - Resize for specs
   - Adjust safe zones
   - Platform-appropriate tone
   - Optimize text size

3. **Create platform variants**
   - Instagram: 1080x1080px
   - Facebook: 1200x630px
   - Twitter: 1200x675px
   - LinkedIn: 1200x627px
   - Pinterest: 1000x1500px
   - TikTok: 1080x1920px

4. **Maintain consistency**
   - Same core message
   - Recognizable brand elements
   - Adapted, not duplicated
   - Platform-native feel

## Design Principles for Social

**Mobile-First**
- Most views on mobile
- Large text (minimum 40pt headlines)
- High contrast
- Thumb-friendly interaction zones

**Scroll-Stopping**
- First impression in 0.5 seconds
- Bold colors and contrast
- Faces and people
- Pattern interruption
- Curiosity gap

**Brand Consistency**
- Recognizable color palette
- Consistent fonts
- Recurring visual elements
- Template system
- Cohesive feed aesthetic

**Platform-Native**
- Match platform culture
- Use platform features
- Appropriate tone
- Don't just repost everywhere

**Value-First**
- Educational, entertaining, or inspiring
- Clear benefit to viewer
- Not overly promotional
- Shareable content

## Typography for Social Media

**Headline Sizes**
- Instagram feed: 60-80pt
- Instagram story: 80-120pt
- LinkedIn: 48-72pt
- Twitter: 72-96pt

**Font Choices**
- Bold, high-contrast fonts
- Sans-serif for readability
- Avoid thin weights
- Limit to 2 fonts
- Consistent hierarchy

**Text Amount**
- Instagram feed: Minimal (image speaks)
- Instagram story: Short, scannable
- LinkedIn: Can be text-heavy
- Twitter: Concise, punchy
- Pinterest: Descriptive headlines

## Color Strategy

**High-Contrast Palettes**
- Stand out in crowded feeds
- Improve mobile readability
- Create visual hierarchy
- Aid accessibility

**Brand Colors**
- Primary color prominently
- Accent for highlights
- Consistent background treatment
- Recognize at glance

**Platform Considerations**
- Instagram: Bright, saturated
- LinkedIn: Professional, muted
- Twitter: Bold, attention-grabbing
- Pinterest: Soft, aspirational

## Best Practices

- **Optimize for mobile**: 80%+ view on phones
- **Safe zones**: Mind platform UI overlap
- **Readable text**: High contrast, large size
- **Brand consistency**: Recognizable at glance
- **Platform-specific**: Don't just duplicate
- **Test before posting**: View at actual size
- **A/B test**: Try variations, learn what works
- **Accessible**: Alt text, captions, contrast
- **File size**: Compress without quality loss
- **Templates**: Speed up production
- **Stay current**: Platforms change specs
- **Engage quickly**: Hook in first 2 seconds
