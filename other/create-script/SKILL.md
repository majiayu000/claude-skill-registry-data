# Create Script Skill

숏폼 영상 스크립트를 생성하는 스킬입니다.

## 사용법

```
/create-script [연예인명]
또는
/create-script [리서치파일경로]
```

## 실행 프로세스

### Step 1: 데이터 로드
1. `data/research/[연예인명]_*.json` 파일 탐색
2. 검증된 제품 정보 우선 선별
3. TOP 3 제품 선정 (다양한 카테고리)

### Step 2: 후킹 작성 (0-3초)
후킹 유형 선택:
```
1. 질문형: "OOO이 매일 쓰는 화장품 궁금하지 않아?"
2. 놀람형: "이 제품 3만원인데 OOO이 5통째 쓰는 중"
3. 공감형: "OOO 피부 비결 찾았다"
4. 반전형: "OOO은 이 비싼 제품 안 써요"
```

### Step 3: 본문 작성 (3-53초)
구성:
- 연예인 소개 (5초)
- 제품 1 소개 (15초)
- 제품 2 소개 (15초)
- 제품 3 소개 (15초)

제품 소개 포인트:
- 제품명 + 브랜드
- 가격대
- 특징/효과 한 줄
- 출처 언급

### Step 4: CTA 작성 (53-60초)
```
- "더 보려면 팔로우!"
- "저장해두고 나중에 사세요"
- "알고 싶은 연예인 댓글로!"
```

## 출력 형식

```markdown
# [연예인명] 실제 사용 화장품 TOP 3

- 예상 길이: 60초
- 타겟 플랫폼: TikTok / Instagram Reels
- 톤앤매너: 친근하고 정보 전달형

---

## 후킹 (0:00-0:03)
**대본:**
> [후킹 멘트]

**화면:**
- [화면 구성 설명]

---

## 연예인 소개 (0:03-0:08)
**대본:**
> [연예인 소개 멘트]

**화면:**
- [연예인 사진/영상]
- 자막: [이름, 직업]

---

## 제품 1: [제품명] (0:08-0:23)
**제품 정보:**
- 브랜드: [브랜드명]
- 가격: [가격]
- 출처: [출처]

**대본:**
> [제품 소개 멘트]

**화면:**
- [제품 이미지]
- 자막: [제품명 + 가격]

---

## 제품 2: [제품명] (0:23-0:38)
[동일 구조]

---

## 제품 3: [제품명] (0:38-0:53)
[동일 구조]

---

## CTA (0:53-1:00)
**대본:**
> [CTA 멘트]

**화면:**
- 팔로우 버튼 강조
- 저장 아이콘 애니메이션

---

## 제작 노트
- BGM 추천: [곡명/분위기]
- 자막 스타일: [스타일 설명]
- 출처 표기: 영상 설명란에 포함
```

## 작성 원칙

1. **구어체 사용**: 친근하고 자연스러운 말투
2. **짧은 문장**: 한 문장 3초 이내
3. **숫자 강조**: 가격, 순위 등 시각적 강조
4. **출처 명시**: 신뢰도를 위해 출처 언급
