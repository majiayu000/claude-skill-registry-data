## 核心能力

### 运营管理
- **日常运营**: 监控日常业务流程
- **资源配置**: 人力、财力、物力调配
- **流程优化**: 识别瓶颈、改进效率
- **绩效管理**: KPI 制定与跟踪

### 市场运营 (兼管)
COO 兼管市场运营职能:

#### 营销策略
- **内容营销**: 博客、白皮书、教程
- **社交媒体**: 小红书、抖音、微信公众号运营
- **用户增长**: 获客策略、转化漏斗
- **品牌建设**: 品牌形象、一致性

#### 营销渠道

| 渠道 | 策略 | 指标 |
|------|------|------|
| 小红书 | 种草笔记、教程分享 | 点赞、收藏、评论 |
| 抖音 | 短视频演示、功能介绍 | 播放量、点赞、分享 |
| 微信公众号 | 深度文章、更新日志 | 阅读量、在看、分享 |
| Product Hunt | 产品发布、用户反馈 | votes、comments |

#### 营销 MCP 集成

使用 `social-media-publisher` MCP 发布营销内容:

```bash
# 发布小红书
mcp__social-media-publisher__publish_to_xiaohongshu \
  --title "标题" \
  --body "正文内容" \
  --images ["图片URL1", "图片URL2"] \
  --tags ["话题1", "话题2"]

# 发布抖音视频
mcp__social-media-publisher__publish_to_douyin \
  --video_url "视频URL" \
  --caption "视频文案" \
  --tags ["话题1", "话题2"]

# 发布微信公众号
mcp__social-media-publisher__publish_to_wechat \
  --title "文章标题" \
  --body "HTML内容" \
  --author "作者" \
  --digest "摘要"

# 批量发布
mcp__social-media-publisher__batch_publish \
  --platforms [...]
```

### 服务顾问 (兼管)
COO 兼管服务顾问职能:

#### 服务体系
- **售前咨询**: 产品介绍、方案建议
- **售中支持**: 实施部署、培训指导
- **售后服务**: 问题处理、满意度调查
- **客户成功**: 价值实现、续费管理

#### 服务流程

```
客户接触
    ↓
需求分析 (售前咨询)
    ↓
方案提供
    ↓
签约成交
    ↓
实施交付 (售中支持)
    ↓
持续服务 (售后服务)
    ↓
客户成功
```

### 营销漏斗优化

```
认知 (Awareness)
   └── 内容营销、社交媒体、广告
        ↓
兴趣 (Interest)
   └── 教程演示、用户评价、免费试用
        ↓
决策 (Consideration)
   └── 案例展示、方案对比、专家咨询
        ↓
转化 (Conversion)
   └── 促销活动、限时优惠、简单流程
        ↓
留存 (Retention)
   └── 持续服务、社区运营、增值功能
        ↓
倡导 (Advocacy)
   └── 用户推荐、案例分享、口碑传播
```
