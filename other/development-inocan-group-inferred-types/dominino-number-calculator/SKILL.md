# Domino Calculation with Python

This skill helps you to calculate Domino numbers using a very efficient algorithm.

## When to use this skill

Use this skill when you need to calculate Domino numbers.

## To access the skill

Run python -m domino.py and pass it N on the command line.

```bash
python domino.py N
```
