# Brevard County Site Development Skill

## When to Invoke

- User asks about developing property in Brevard County
- Zoning questions for Brevard County parcels
- Permit process questions
- Setback or environmental questions

## 12-Stage Pipeline

```
1. Discovery      → Identify parcel, ownership
2. Parcel Data    → BCPAO data, boundaries
3. Zoning         → Zoning code, allowed uses
4. Setbacks       → Calculate buildable area
5. Utilities      → Water, sewer, electric
6. Environmental  → SJRWMD, flood, wetlands
7. Survey         → Boundary, topographic
8. Engineering    → Site engineering
9. Permit Prep    → Application package
10. Submission    → Submit to county
11. Review        → Track, respond to comments
12. Approval      → Final approval
```

## Brevard County Zoning Quick Reference

### Residential
- RS-1 through RS-6: Single-family (decreasing lot size)
- RM-1 through RM-3: Multi-family

### Commercial
- BU-1: Limited retail
- BU-2: General commercial
- BU-1-A: Neighborhood commercial

### Industrial
- IU: Light industrial
- IU-1: Heavy industrial
