# Jira Code Review

Review code changes associated with a Jira issue.

## Usage

```
/jira:review <issue-key>
```

## Features

- Reviews linked commits
- Checks PR quality
- Validates tests
- Verifies acceptance criteria
- Provides feedback
- Updates Jira status

## Related Commands

- `/jira:pr` - Create pull request
- `/jira:work` - Work on issue
