# Travel Planning

## Overview

This skill provides guidance on structuring travel itineraries for optimal experience. It covers pacing guidelines, activity sequencing, and practical considerations for multi-day trips.

## Usage

The main agent uses this skill when:
- Creating initial itinerary drafts
- Balancing activity intensity across days
- Structuring travel days vs. activity days
- Formatting final deliverables

## Pacing Guidelines

- **First day:** Light activities, account for jet lag and travel fatigue
- **Big activity days:** One major attraction + downtime + casual evening
- **Travel days:** Moving between cities counts as the day's activity
- **Last day:** Keep flexible for last-minute shopping or revisits

## Itinerary Structure

Each day should include:
- Morning activity (optional early start)
- Lunch recommendation near morning location
- Afternoon activity or downtime
- Dinner recommendation
- Evening option (optional)
