# Research with Citations Skill

Conduct research with proper citations, source verification, and academic standards.

## When to Use
- Academic writing
- Professional reports
- Fact-checking
- Literature reviews

## Core Capabilities
- Source discovery
- Citation formatting (APA, MLA, Chicago)
- Credibility assessment
- Bibliography generation
- Plagiarism detection
- Cross-referencing

## Citation Formats
```
APA: Author, A. A. (Year). Title. Publisher.
MLA: Author. "Title." Publisher, Year.
Chicago: Author. Title. Publisher, Year.
```

## Source Evaluation (CRAAP Test)
- **Currency**: How recent?
- **Relevance**: Meets your needs?
- **Authority**: Who wrote it?
- **Accuracy**: Evidence-based?
- **Purpose**: Why does it exist?

## Tools
- Zotero: Reference management
- Google Scholar: Academic search
- Connected Papers: Literature mapping
- Crossref: DOI lookup

## Best Practices
- Use primary sources
- Verify facts with multiple sources
- Track all sources during research
- Use citation manager
- Avoid citation mills
