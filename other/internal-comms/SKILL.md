## When to use this skill
要编写内部通信，请将此技能用于：
- 3P 更新（Progress、Plans、Problems）
- 公司通讯
- 常见问题解答
- 状态报告
- 领导更新
- 项目更新
- 事件报告

## How to use this skill

要编写任何内部通信：

1. **识别通信类型** 从请求中
2. **加载相应的指南文件** 从 `examples/` 目录：
    - `examples/3p-updates.md` - 用于 Progress/Plans/Problems 团队更新
    - `examples/company-newsletter.md` - 用于全公司通讯
    - `examples/faq-answers.md` - 用于回答常见问题
    - `examples/general-comms.md` - 用于其他不明确匹配上述任何一项的内容
3. **遵循该文件中的具体说明** 关于格式、语气和内容收集

如果通信类型与任何现有指南不匹配，请询问澄清或更多关于所需格式的上下文。
