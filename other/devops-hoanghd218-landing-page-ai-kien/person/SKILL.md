## RULES

### MUST Include
- Verified property ownership
- Corporate property holdings if connected
- High-value assets if publicly documented
- Source for all asset claims

### MUST NOT
- State ownership without verification
- Include estimated values without qualifier
- Speculate on hidden assets
- Include private financial information without source
