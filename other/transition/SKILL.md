# GSD Transition Workflow

Workflow for managing project transitions.

## When to Use

- Transitioning between phases
- Handing off work
- Changing context

## Phases

1. Document current state
2. Prepare transition
3. Execute handover
4. Verify readiness

## Success Criteria

Transition completed successfully.
