# Скилл «Бевзенко»: Глубокие юридические заключения

## Назначение

Создание профессиональных юридических заключений с академической глубиной анализа, многоуровневой аргументацией и солидной доктринальной базой.

## Структура заключения

### Обязательные разделы

1. **Титульный лист**: название, автор (степень, должность), дата, место
2. **Описание ситуации**: нумерованные пункты, хронология, ключевые факты
3. **Вопросы**: чётко сформулированные правовые вопросы
4. **Краткие ответы**: резюме выводов (1–2 абзаца на вопрос)
5. **Развёрнутые ответы**: полная аргументация по разделам A, B, C...
6. **Сноски**: библиография в академическом формате

### Принцип многоуровневой защиты

Каждый тезис защищать несколькими альтернативными аргументами:
- Основной аргумент (если принят — дело выиграно)
- Субсидиарный аргумент (если основной отклонён)
- Запасной аргумент (если оба предыдущих не сработали)

Формулировка: «Даже если считать, что [контраргумент], то [альтернативный довод]...»

## Методология аргументации

### Доктринальный фундамент

Строить аргументацию на авторитетных источниках:
- Классическая цивилистика (германская, французская школы)
- Дореволюционная русская доктрина
- Советская цивилистическая школа
- Современные российские учёные
- Актуальная судебная практика (ВС РФ, КС РФ)

### Структура аргумента

1. **Тезис**: чёткая формулировка позиции
2. **Теоретическое обоснование**: доктринальные источники
3. **Нормативное подтверждение**: ссылки на статьи кодексов
4. **Практическое подкрепление**: судебные акты
5. **Логическое завершение**: вывод для конкретного случая

### Работа с конкурирующими теориями

При наличии научной дискуссии:
1. Изложить обе позиции объективно
2. Показать историю развития спора
3. Указать победившую точку зрения
4. Обосновать выбор позиции для заключения

## Стилистические требования

### Терминология

- Использовать точные юридические термины
- Латинские термины курсивом: *res judicata*, *ius exceptionis*
- Последовательность: выбранный термин использовать единообразно

### Синтаксис

- Предложения: не более 35 слов
- Избегать цепочек вводных конструкций
- Одна мысль — один абзац

### Форматирование

- Иерархическая нумерация разделов (I, II, III → A, B, C → 1, 2, 3)
- Заголовки разделов — тезисы («Иски о признании не подлежат давности»)
- Сноски внизу страницы с полными библиографическими данными

## Шаблон документа

```javascript
const { Document, Packer, Paragraph, TextRun, AlignmentType, FootnoteReferenceRun } = require('docx');

const doc = new Document({
  footnotes: {
    1: {
      children: [
        new Paragraph({
          children: [
            new TextRun({
              text: "Покровский И.А. Основные проблемы гражданского права. М., 1917. С. 245.",
              size: 20
            })
          ]
        })
      ]
    }
  },
  sections: [{
    children: [
      // Титул
      new Paragraph({
        alignment: AlignmentType.CENTER,
        spacing: { after: 400 },
        children: [
          new TextRun({
            text: "ПРАВОВОЕ ЗАКЛЮЧЕНИЕ",
            bold: true,
            size: 32
          })
        ]
      }),
      
      new Paragraph({
        alignment: AlignmentType.CENTER,
        children: [
          new TextRun({
            text: "по вопросу о [предмет заключения]",
            size: 24
          })
        ]
      }),
      
      // Раздел с тезисом-заголовком
      new Paragraph({
        spacing: { before: 400, after: 200 },
        children: [
          new TextRun({
            text: "A. Иски о признании права не подлежат исковой давности",
            bold: true,
            size: 26
          })
        ]
      }),
      
      // Параграф со сноской
      new Paragraph({
        alignment: AlignmentType.JUSTIFIED,
        children: [
          new TextRun({
            text: "Как справедливо отмечал И.А. Покровский, ",
            size: 24
          }),
          new FootnoteReferenceRun(1),
          new TextRun({
            text: " институт исковой давности имеет своей целью...",
            size: 24
          })
        ]
      })
    ]
  }]
});
```

## Оформление ссылок

### Научная литература

```
Автор И.О. Название работы. Город: Издательство, Год. С. XX.
Автор И.О. Название статьи // Журнал. Год. № X. С. XX–XX.
```

### Судебная практика

```
Определение ВС РФ от DD.MM.YYYY № XXX-ЭС00-00000
Постановление Пленума ВС РФ от DD.MM.YYYY № XX «О...»
```

### Нормативные акты

```
Статья XX Гражданского кодекса Российской Федерации
Пункт X статьи XX Федерального закона от DD.MM.YYYY № XXX-ФЗ «О...»
```
