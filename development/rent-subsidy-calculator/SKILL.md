# 租金補貼計算技能

基於「300億元中央擴大租金補貼專案計畫 2.0」的精確計算器。

## 快速開始

執行計算：
```bash
node .claude/skills/租金補貼計算/scripts/calculate.mjs --json '{"age":28,"rentalCity":"臺北市","monthlyIncome":42000,"ethnicity":"阿美族"}'
```

## 必要資訊收集

在計算前，務必向使用者詢問以下資訊：

| 欄位 | 說明 | 必填 |
|------|------|------|
| `age` | 年齡 | ✓ |
| `rentalCity` | 租屋縣市（非戶籍地） | ✓ |
| `monthlyIncome` | 月薪（元） | ✓ |
| `familyMembers` | 家庭成員人數 | 預設 1 |
| `isMarried` | 是否已婚 | 預設 false |
| `marriageYears` | 結婚年數 | 新婚加碼用 |
| `minorChildren` | 未成年子女人數（含胎兒） | 預設 0 |
| `ethnicity` | 族群（如：阿美族、泰雅族） | 原住民加碼用 |
| `isLowIncome` | 是否低收入戶 | 預設 false |
| `isMidLowIncome` | 是否中低收入戶 | 預設 false |
| `isDisabled` | 是否身心障礙 | 預設 false |

## 計算流程（嚴格遵守）

### Step 1: 資格檢核
```
✓ 中華民國國民且設籍國內
✓ 年滿 18 歲
✓ 家庭成員無自有住宅
✓ 已在租屋中
✓ 平均月所得 ≤ 租賃地所得門檻
```

### Step 2: 級距判定（決定基本金額）

```
第一級（最高）:
  - 家庭成員 ≥ 2 人 且 低收入戶
  - 或：家庭成員 ≥ 3 人 且 中低收入戶

第三級（最低）:
  - 家庭成員 = 1 人（單身）
  - 且：年齡 < 40 歲
  - 且：非經濟弱勢（非低收、非中低收）
  - 且：非社會弱勢（非原住民、非身障、非 65 歲以上等）
  ⚠️ 原住民是社會弱勢，不適用第三級！

第二級（中間）:
  - 不屬於第一級 且 不屬於第三級
```

### Step 3: 加碼倍數（取最高者，不疊加）

| 優先序 | 身份 | 倍數 |
|--------|------|------|
| 1 | 育有未成年子女 3 人以上 | 1.8 倍 |
| 2 | 育有未成年子女 2 人 | 1.6 倍 |
| 3 | 育有未成年子女 1 人（含胎兒） | 1.4 倍 |
| 4 | 低收入戶 或 中低收入戶 | 1.4 倍 |
| 5 | 結婚 2 年內新婚家庭 | 1.3 倍 |
| 6 | 18-39 歲單身青年 | 1.2 倍 |
| 7 | 社會弱勢（原住民、身障、65歲以上等） | 1.2 倍 |
| 99 | 無加碼 | 1.0 倍 |

### Step 4: 計算公式

```
最終補貼 = 基本金額（依級距和縣市）× 加碼倍數（取最高者）
```

## 常見地雷警告

### 地雷 1：原住民誤判為第三級
```
❌ 錯誤：28歲單身阿美族 → 第三級
✓ 正確：28歲單身阿美族 → 第二級（原住民是社會弱勢）
```

### 地雷 2：加碼倍數疊加
```
❌ 錯誤：原住民 1.2 倍 + 青年 1.2 倍 = 1.44 倍
✓ 正確：取最高者 1.2 倍（不疊加）
```

### 地雷 3：混淆經濟弱勢與社會弱勢
```
經濟弱勢（1.4倍）：低收入戶、中低收入戶
社會弱勢（1.2倍）：原住民、身障、65歲以上等
❌ 原住民 ≠ 低收入戶
```

### 地雷 4：使用戶籍地計算
```
❌ 錯誤：依「戶籍地」計算補貼金額
✓ 正確：依「租賃地」計算補貼金額
```

## 社會弱勢類型清單

- 原住民（16 族）
- 身心障礙者
- 65 歲以上
- 特殊境遇家庭
- 安置機構結束安置未滿 25 歲
- 家暴或性侵受害者及其子女
- HIV 感染者或 AIDS 患者
- 災民
- 遊民
- 因懷孕或生育遭遇困境未成年人

## 原住民族清單（16 族）

阿美族、泰雅族、排灣族、布農族、卑南族、魯凱族、鄒族、賽夏族、雅美族（達悟族）、邵族、噶瑪蘭族、太魯閣族、撒奇萊雅族、賽德克族、拉阿魯哇族、卡那卡那富族

## 執行範例

### 範例 1：28歲阿美族單身在臺北租屋，月薪 42,000
```bash
node .claude/skills/租金補貼計算/scripts/calculate.mjs --json '{
  "age": 28,
  "rentalCity": "臺北市",
  "monthlyIncome": 42000,
  "familyMembers": 1,
  "ethnicity": "阿美族",
  "isMarried": false
}'
```

預期結果：
- 級距：第二級（原住民是社會弱勢，不適用第三級）
- 基本金額：5,000 元
- 加碼：1.2 倍（原住民/青年，取較高者）
- 最終補貼：5,000 × 1.2 = 6,000 元/月

### 範例 2：35歲新婚育有 1 子在新北租屋，月薪 50,000
```bash
node .claude/skills/租金補貼計算/scripts/calculate.mjs --json '{
  "age": 35,
  "rentalCity": "新北市",
  "monthlyIncome": 50000,
  "familyMembers": 3,
  "isMarried": true,
  "marriageYears": 1,
  "minorChildren": 1
}'
```

預期結果：
- 級距：第二級
- 基本金額：4,000 元
- 加碼：1.4 倍（育有 1 子，高於新婚 1.3 倍）
- 最終補貼：4,000 × 1.4 = 5,600 元/月
