# 窝窝乐七言律诗生成器

创作一首标准七言律诗，严格遵守格律。

## 格式要求

- **总计**：8行，每行7字，共56字
- **韵脚**：第2、4、6、8行押平声韵（用"e"韵：乐、客、色、迫等）
- **平仄**：首句可拗，次句起严格对仗
- **对仗**：第3-4、5-6行严格对仗

## 主题：窝窝乐

描绘可爱、暴躁、搞笑的"窝窝乐"形象：
- 外表软萌可爱
- 内心暴躁火爆
- 时而发怒时而卖萌
- 让人又爱又怕

## 创作示例

```
窝窝乐兮性情狂，（首联）
怒目圆睁气势昂。
软萌外表藏火焰，（颔联-对仗）
暴躁心中蕴柔光。
时而咆哮惊四座，（颈联-对仗）
时而撒娇醉八方。
世人皆叹奇妙处，（尾联）
爱恨交织意难忘。
```
