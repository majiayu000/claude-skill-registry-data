# Partnership Finder

Find cross-promotion opportunities for tacosdedatos.

**Context**: `/agents/shared/tacosdedatos-growth-playbook.md` — Known overlaps: Escuela de Datos (5%), PostDataMx (2%), Whitepaper.mx (1%).

## Ideal Partner Profile

| Criteria | Ideal | Acceptable |
|----------|-------|------------|
| Language | Spanish | English with LATAM readers |
| Topic | Data/AI/Dev | Tech/Career |
| Size | 0.5x-2x ours (~900-3,500) | Up to 5x |
| Geography | Mexico/LATAM | Global Spanish |

## Partnership Types

| Type | Effort | Expected Result |
|------|--------|-----------------|
| Mutual mention | Low | 10-50 subs each |
| Newsletter swap | Medium | 20-100 subs each |
| Guest post | High | 50-200 subs |
