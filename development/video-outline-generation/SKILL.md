# 视频大纲生成

## 何时使用此Skill
当用户出现以下任一需求时，使用此技能：
- 需要制作"视频大纲"、"视频脚本"
- 进行"视频策划"、"视频内容规划"
- 准备"拍视频"、"做视频"
- 需要"视频创意"、"视频方向"
- 想要"视频结构"、"内容框架"

## 核心目标
根据用户需求快速生成多个视频创作方案，每个方案都包含完整的大纲结构，帮助用户快速开始视频制作。

## 执行流程

### 第一步：需求分析和定位确认
**目标**：明确视频的目标、受众和风格

**关键要素分析**：
1. **视频目标定位**
   - 主要目的：知识传递、娱乐搞笑、产品推广、品牌建设
   - 成功指标：观看量、点赞率、完播率、转化率
   - 期望效果：教育启发、情感共鸣、商业转化

2. **目标受众分析**
   - 年龄段：青少年、青年、中年、老年
   - 兴趣偏好：科技、生活、教育、娱乐
   - 观看习惯：短视频(1-3分钟)、中等(5-10分钟)、长视频(10分钟+)
   - 消费场景：碎片时间、学习时间、休闲时间

3. **内容类型确认**
   - 知识科普：教程、讲解、分析
   - 生活分享：vlog、日常、体验
   - 产品评测：开箱、测试、对比
   - 娱乐内容：搞笑、挑战、故事

4. **资源约束评估**
   - 出镜意愿：真人出镜、画外音、纯字幕
   - 拍摄设备：手机、相机、屏幕录制
   - 场景限制：室内、户外、特定环境
   - 制作能力：剪辑、特效、动画

### 第二步：多方案大纲设计
**目标**：生成2-3个不同风格的视频方案

#### 方案1：[视频标题] - [方案类型]
**视频定位**：[知识分享型/生活体验型/产品评测型/娱乐互动型]

**预计时长**：[X分钟]
**目标完播率**：[X%]
**核心亮点**：
- 🎯 [主要吸引力1]
- 💡 [创新点1]
- 📈 [传播优势1]

**缩略图设计建议**：
- **主体元素**：[画面的主要内容]
- **文字设计**：[标题文字的处理方式]
- **色彩搭配**：[主色调和配色方案]
- **表情/动作**：[人物的表情或动作建议]
- **构图风格**：[整体构图和视觉风格]

**视频大纲结构**：

```
🎬 开头 (0-15秒) - 黄金钩子
├── 悬念引入：[如何在前3秒抓住注意力]
├── 价值承诺：[告诉观众能获得什么]
└── 人设建立：[建立与观众的情感连接]

📺 主体部分 (15秒-X分钟) - 核心内容
├── 第一部分：[主要内容和结构]
│   ├── 要点1：[具体内容]
│   ├── 要点2：[具体内容]
│   └── 要点3：[具体内容]
├── 第二部分：[递进内容]
│   ├── 深入分析：[详细说明]
│   ├── 实际演示：[操作或展示]
│   └── 效果展示：[结果或反馈]
└── 第三部分：[升华内容]
    ├── 总结要点：[归纳重点]
    ├── 延伸思考：[启发观众]
    └── 实用建议：[可操作的建议]

🎭 结尾 (最后15秒) - 互动收尾
├── 价值回顾：[重申核心价值]
├── 互动引导：[点赞、评论、关注]
└── 下期预告：[建立期待感]
```

**拍摄制作要点**：
- **场景布置**：[背景和环境的建议]
- **设备要求**：[拍摄设备和技术要求]
- **表演指导**：[出镜时的表现技巧]
- **剪辑重点**：[后期制作的关键点]

**优势分析**：
- ✅ [优势1：如制作难度适中]
- ✅ [优势2：如内容价值突出]
- ✅ [优势3：如容易传播]

**挑战预估**：
- ⚠️ [挑战1：如需要特定设备]
- ⚠️ [挑战2：如表演难度较高]
- ⚠️ [挑战3：如内容专业性强]

### 第三步：具体内容填充
**目标**：为每个大纲部分提供详细的内容建议

**内容填充要点**：

#### 开头部分详细设计
```
3秒钩子示例：
- 悬念型："我发现了一个让我震惊的秘密..."
- 数据型："90%的人都不知道这个技巧..."
- 结果型："我用10天时间做到了这件事..."
- 情感型："那一刻我真的哭了..."

15秒价值承诺：
- 明确告知观众能获得什么
- 用具体数字量化价值
- 建立紧迫感和稀缺性
```

#### 主体部分详细设计
```
内容组织方式：
1. 问题-解决方案结构
   - 提出普遍问题
   - 分析问题原因
   - 提供解决方案
   - 展示实施效果

2. 时间顺序结构
   - 过去的状态
   - 转折点事件
   - 改变的过程
   - 现在的结果

3. 对比分析结构
   - 错误的做法
   - 正确的做法
   - 效果对比
   - 选择建议
```

#### 结尾部分详细设计
```
互动引导语：
- "你们有什么想法？评论区告诉我～"
- "点赞收藏，下次不迷路！"
- "关注我，每天分享有用内容"
- "下期想看什么？在评论区告诉我！"

CTA设计：
- 明确的行动指令
- 低的行动门槛
- 即时的行动激励
```

### 第四步：多维度对比推荐
**目标**：帮助用户选择最适合的方案

**对比维度**：
1. **制作难度**：设备要求、技能要求、时间投入
2. **内容价值**：实用性、独特性、专业性
3. **传播潜力**：话题性、分享性、传播度
4. **个人匹配**：兴趣匹配、能力匹配、资源匹配

**推荐逻辑**：
```
如果你是 [新手/有经验/专业创作者]，推荐 [方案X]
因为 [制作难度/内容价值/传播潜力] 更适合你。

如果你想 [快速上手/做出精品/追求流量]，推荐 [方案Y]
因为它在 [制作成本/内容质量/传播效果] 方面有优势。
```

## 特殊场景处理

### 1. 零基础创作者
**推荐策略**：
- 优先推荐操作简单、成本低的方案
- 选择不需要出镜或对表演要求低的内容
- 提供详细的制作指导和技巧

### 2. 追求高质量内容
**推荐策略**：
- 推荐需要一定设备和技术要求的方案
- 注重内容的专业性和深度
- 提供高质量的制作标准

### 3. 商业推广需求
**推荐策略**：
- 推荐产品评测或体验类内容
- 注重转化率和商业价值
- 平衡内容质量和商业目标

## 质量标准

### 大纲质量标准
- [ ] 结构清晰，逻辑连贯
- [ ] 内容价值明确，有实用性
- [ ] 符合平台特点和受众偏好
- [ ] 在用户能力范围内可实现

### 创意质量标准
- [ ] 标题吸引人，有点击欲
- [ ] 开头有钩子，能抓住注意力
- [ ] 内容有亮点，能产生共鸣
- [ ] 结尾有引导，能促进互动

## 交付示例

### 示例：效率提升视频大纲

**需求**：制作一个关于工作效率提升的视频

**生成方案**：

#### 方案1：3个让你效率翻倍的时间管理技巧
**预计时长**：8分钟
**缩略图**：时钟+火箭，文字"效率翻倍"

**大纲结构**：
```
开头(15秒)：
"每天都很忙，但总觉得什么都没做完？今天分享3个让我效率翻倍的技巧！"

主体部分(7分钟)：
1. 番茄工作法 - 专注25分钟，休息5分钟
2. 时间块管理 - 把任务安排到固定时间段
3. 留白时间 - 每天留1小时应对突发情况

结尾(30秒)：
"试试这些方法，告诉我哪个对你最有用！点赞收藏，一起进步！"
```

#### 方案2：我如何用一周时间完成一个月的工作
**预计时长**：10分钟
**缩略图**：对比图"1周 vs 1个月"，惊讶表情

**大纲结构**：
```
开头(20秒)：
"上周我只工作了5天，却完成了平时一个月的工作量。秘密就是..."

主体部分(9分钟)：
1. 发现问题：分析时间黑洞
2. 找到方法：深度工作+批量处理
3. 实施过程：具体的操作步骤
4. 效果验证：数据对比展示

结尾(40秒)：
"效率不是忙出来的，而是巧出来的。你有什么效率秘诀吗？"
```
