# Plugin Updater

インストール済みのプラグインを最新バージョンに更新します。

## ワークフロー

### 1. プラグイン名の確認

ユーザーがプラグイン名を指定していない場合、更新可能なプラグインの一覧を表示して選択を促す。

更新可能なプラグインを確認するには:
1. `.claude-plugin/marketplace.json` から最新バージョンを取得
2. インストール済みプラグインの現在バージョンと比較
3. バージョンが異なるプラグインを一覧表示

### 2. 更新の実行

```bash
claude plugin update [plugin-name]
```

### 3. 結果の報告

更新の結果をユーザーに報告する。

## 出力フォーマット

```
## プラグインの更新

プラグイン `shiiman-xxx` を更新しています...

✅ 更新完了

バージョン: 1.0.0 → 2.0.0

※ 更新を適用するには再起動が必要です。
```

## 重要な注意事項

- ✅ `claude plugin update` コマンドを使用
- ✅ 更新前に現在バージョンと最新バージョンを比較
- ✅ 更新結果をユーザーに報告
- ✅ 再起動が必要な場合はその旨を通知
- ❌ インストールされていないプラグインを指定された場合はエラーメッセージを表示
- ❌ 既に最新バージョンの場合はその旨を通知
