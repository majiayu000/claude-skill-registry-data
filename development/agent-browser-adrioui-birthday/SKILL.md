# Agent Browser Skill

A skill for discovery and implementation with browser-based verification.

## Overview

This skill provides a workflow for discovering code patterns and implementing features with browser-based verification.

## Key Concepts

- **Discovery phase**: Explore codebase to understand patterns and conventions
- **Implementation phase**: Build features using discovered patterns
- **Snapshot references**: Use browser snapshots for verification

## Usage

1. **Discovery**: Use the discovery-session template to explore
2. **Handoff**: Use handoff-note template when switching phases
3. **Implementation**: Use implementation-session template to build
