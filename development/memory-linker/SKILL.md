# Memory Linker

Use this skill to build cross-memory links.

## Workflow

1) Identify related entities and contexts.
2) Link memories with relation type and strength.
3) Validate retrieval improves recall.
4) Store linkage summary.

## Output expectations

- Provide linked entities and relationship rationale.
