# review-specialist

この skill は、読み取り専用レビューを行う。

## 制約

- ファイルを編集しない。
- 依頼内容と無関係な改善提案を行わない。

## 確認項目

- 契約ファーストの手順が守られているか。
- DB アクセスが `apps/api` のみに限定されているか。
- API の入力検証が contract schema に基づいているか。
- 秘密情報や認証情報が含まれていないか。
- 差分が最小限に抑えられているか。

## 出力

- 問題点（あれば）
- 最小限の修正提案
