# コトネちゃん 会社用スキル（StudioJinsei）

## 概要

**StudioJinsei** の公式LINEアカウントで使用するコンテンツを作成します。
プロフェッショナルでありながら、親しみやすさも兼ね備えたデザインを目指します。

## 会社情報

| 項目 | 設定 |
|------|------|
| **会社名** | StudioJinsei（スタジオ人生） |
| **代表** | コトネちゃん |
| **事業** | LINE公式アカウント運用代行 |
| **デザイン哲学** | Apple風ミニマル、余白と洗練 |

## 画風・スタイル

### 手書き風キレイめスタイル

| 項目 | 設定 |
|------|------|
| **絵柄** | 手書き風イラスト、柔らかいタッチ |
| **線** | 柔らかい、少しラフ、温かみ |
| **色味** | 落ち着いた、洗練された、トーン控えめ |
| **表情** | 穏やかな微笑み、安心感、信頼感 |
| **雰囲気** | プロフェッショナル、親しみやすい、上品 |

### デザイン原則（Apple風）

| 原則 | 説明 |
|------|------|
| **余白** | たっぷり取る、呼吸するデザイン |
| **シンプル** | 要素を絞る、ノイズを排除 |
| **洗練** | 高品質、細部までこだわる |
| **一貫性** | カラー・フォント・トーンを統一 |

### カラーパレット（会社用）

| 名前 | カラーコード | 用途 |
|------|-------------|------|
| **ソフトミント** | `#A8D5BA` | メイン（落ち着いた緑） |
| **ダークティール** | `#1D4E4A` | テキスト・見出し |
| **ペールミント** | `#E8F5EE` | 背景 |
| **ホワイト** | `#FFFFFF` | ベース |
| **ウォームグレー** | `#4A4A4A` | 本文テキスト |
| **ライトグレー** | `#F5F5F7` | 背景（Apple風） |

### 背景スタイル
- 白ベース or 薄いミントグラデーション
- 余白たっぷり
- 装飾は最小限
- クリーンでノイズなし

## フォント設定

### 日本語
| 用途 | フォント |
|------|---------|
| 見出し | Zen Maru Gothic (Medium/Bold) |
| 本文 | Noto Sans JP (Regular) |

### 英語
| 用途 | フォント |
|------|---------|
| ロゴ・見出し | Poppins (Medium/SemiBold) |
| 本文 | Inter (Regular) |

## 口調・言葉遣い

### トーン
- 丁寧、プロフェッショナル
- 親しみやすさも忘れない
- 信頼感を与える

### 例文
```
「こんにちは、StudioJinseiです。」
「本日は〇〇についてお伝えします。」
「ご質問がございましたら、お気軽にどうぞ。」
「いつもご利用ありがとうございます。」
「シンプルに、あなたのビジネスをサポートします。」
```
