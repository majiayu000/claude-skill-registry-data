# Use Codex

Codex CLI を使用して各種タスクを実行する。

## 機能

| 機能 | 説明 | スクリプト |
|------|------|-----------|
| Web検索 | Web検索を実行 | `web-search.sh` |
| ファイルレビュー | コードレビューを依頼 | `file-review.sh` |

## 使用方法

詳細は各スクリプトのドキュメントを参照:

- @./web-search.md
- @./file-review.md

## 前提条件

- Codex CLI がインストールされていること
- Codex CLI が適切に設定されていること
