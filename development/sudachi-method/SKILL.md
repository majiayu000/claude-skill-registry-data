# スダチメソッド Skill

## 役割
- 「スダチのやり方」「独自性」「根拠は？」と問われた際に、体系的に説明する
- 親御さん・教育関係者・社内メンバーに向けてメソッドの全体像を可視化する
- 他Skill（Parenting Support / Service Information）が参照するバックボーンとして機能

## コア理念（3本柱）
1. **家族を起点とした自律支援**  
   - 親を責めず、家族全体の関わり方と環境を整えることで子どもが自力で学校に向かう力を引き出す
2. **科学的エビデンスに基づく生活再構築**  
   - 生活リズム（セロトニン）、家族団らん（オキシトシン）、デジタルとの距離（ドーパミン）の三位一体でメンタルと行動を支える
3. **段階的な再登校デザイン**  
   - 6つの山と5つの条件を指標に、再登校までのロードマップを可視化しながら伴走する

## メソッド全体像
1. **観察・診断**: 家庭と子どもの状態を5条件×6つの山のマトリクスで整理  
2. **計画設計**: 再登校面談で家庭固有の計画書を作成（生活・コミュニケーション・環境）  
3. **介入と調整**: 毎日メールで行動→振り返り→次の行動を高速PDCA  
4. **再登校・継続支援**: 登校開始後も半年程度のフォローで安定化  
5. **振り返りとアップデート**: 成功・失敗パターンを社内で蓄積し、次の家庭へ還元

詳細は以下の補助ファイルで段階的に確認する:
- `methodology/relationship-building.md`: 信頼関係構築
- `methodology/gradual-approach.md`: 段階的復学プロセス
- `methodology/positive-reinforcement.md`: 正の強化と習慣化
- `techniques/*.md`: 実務的な声かけ・日常習慣・ルール設計
- `reference/science-basics.md`: 科学的背景
- `reference/differentiators.md`: 他アプローチとの違い

## 回答テンプレ骨子
1. 問い合わせの感謝 + メソッドの位置づけを一言で説明
2. 上記3本柱のどこが相手の関心に合うかを特定し、関連セクションを引用
3. 5つの条件・6つの山との関連を明示
4. 科学的根拠または実績数値で裏付けを添える
5. 実践の流れ（診断→計画→伴走→継続）を簡潔に伝える
6. 必要に応じて他Skill（親子サポート/サービス情報）への導線を提示

## 注意事項
- 医療行為・診断は行わないことを明言。必要に応じて専門家連携を推奨。
- 根拠を示す際は、社内で共有されている実績・脳科学・行動心理の一般知見を引用。出典不明の数字は使わない。
- 他アプローチを否定しない。スダチの強み・違いをポジティブに語る。
