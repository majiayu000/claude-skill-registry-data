# Plugin Disabler

有効なプラグインを無効化します。

## ワークフロー

### 1. プラグイン名の確認

ユーザーがプラグイン名を指定していない場合、有効なプラグインの一覧を表示して選択を促す。

### 2. 無効化の実行

```bash
claude plugin disable [plugin-name]
```

### 3. 結果の報告

無効化の結果をユーザーに報告する。

## 出力フォーマット

```
## プラグインの無効化

プラグイン `shiiman-xxx` を無効化しています...

✅ 無効化完了

プラグイン `shiiman-xxx` が無効になりました。
```

## 重要な注意事項

- ✅ `claude plugin disable` コマンドを使用
- ✅ 無効化前にプラグインのインストール状態を確認
- ✅ 無効化結果をユーザーに報告
- ❌ インストールされていないプラグインを指定された場合はエラーメッセージを表示
- ❌ 既に無効なプラグインを指定された場合はその旨を通知
