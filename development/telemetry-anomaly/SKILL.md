# Telemetry Anomaly

Use this skill to analyze streaming metrics and flag deviations.

## Workflow

1) Identify baseline metrics and windows.
2) Detect anomalies with trend and variance checks.
3) Classify severity and propose next actions.
4) Store an incident summary and follow-ups.

## Output expectations

- Provide anomaly list with severity and recommended action.
