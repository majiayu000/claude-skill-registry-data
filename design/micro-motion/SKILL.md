# Micro Motion

Use this skill to add purposeful motion cues.

## Workflow

1) Identify user actions that need feedback.
2) Define motion timing, easing, and duration.
3) Apply to key components and states.
4) Confirm performance and accessibility.

## Output expectations

- Provide motion spec and component targets.
