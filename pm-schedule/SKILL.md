# スケジュール管理エージェント

## 役割
WBSに基づいてプロジェクトスケジュールを作成し、進捗を管理します。

## スケジュール構成

```
docs/pm/
├── schedule/
│   ├── gantt.md                 # ガントチャート
│   ├── milestones.md            # マイルストーン一覧
│   ├── timeline.md              # タイムライン
│   └── progress/
│       └── YYYY-MM-DD.md        # 進捗報告
└── wbs/                         # WBS（参照）
```

    excludes weekends

    進捗管理             :active,  pm2, 2024-01-08, 60d

    section 3. 設計

    section 4. 実装

## スケジュール管理指標

### EVM (Earned Value Management)

| 指標 | 計算式 | 説明 |
|------|--------|------|
| PV (計画価値) | 予定工数×期間比率 | 今日までの予定進捗 |
| EV (出来高) | 完了工数 | 実際の進捗価値 |
| AC (実コスト) | 実績工数 | 実際にかかった工数 |
| SV (スケジュール差異) | EV - PV | 正=前倒し、負=遅延 |
| CV (コスト差異) | EV - AC | 正=節約、負=超過 |
| SPI (スケジュール効率) | EV / PV | >1=前倒し、<1=遅延 |
| CPI (コスト効率) | EV / AC | >1=効率的、<1=非効率 |

4. **進捗報告**: `docs/pm/schedule/progress/YYYY-MM-DD.md`
