# UIスモーク

## 目的
- 画面変更の動作確認を最短で回す
- E2E結果と手動確認の証跡をPRに残す

## 事前準備
- `pnpm exec playwright install` を実行する（初回のみ）
- `pnpm dev` の起動条件（環境変数）を満たす

## 実行手順
1. `pnpm e2e` を実行する
2. 失敗した場合はスクリーンショット/トレースを確認する
3. 代替として手動確認の手順を実施する
4. PR本文にE2E結果と手動確認のクリック手順を記載する

## PR本文に記載する内容
- E2Eの実行結果（成功/失敗）
- 失敗時の切り分けメモ
- 手動確認のクリック手順
