# Tool Persistence

Use this skill to convert solutions into reusable tools.

## Workflow

1) Identify repeatable logic and inputs.
2) Create script or tool scaffold.
3) Document usage and examples.
4) Store and register the tool.

## Output expectations

- Provide tool path, inputs, and usage instructions.
