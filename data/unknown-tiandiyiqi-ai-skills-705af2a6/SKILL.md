# 竞争对手广告提取器

此技能从广告库提取竞争对手的广告并分析有效的内容——他们强调的问题、针对的用例以及引起共鸣的文案/创意。

## 何时使用此技能

- 研究竞争对手的广告策略
- 为您自己的广告寻找灵感
- 了解市场定位
- 识别成功的广告模式
- 分析有效的 messaging
- 发现新的用例或痛点
- 使用经过验证的概念规划广告活动

## 此技能的功能

1. **提取广告**：从Facebook广告库、LinkedIn等抓取广告
2. **捕获截图**：保存所有广告的视觉副本
3. **分析信息传递**：识别问题、用例和价值主张
4. **对广告进行分类**：按主题、受众或格式分组
5. **识别模式**：找到常见的成功方法
6. **提供洞察**：解释为什么某些广告可能表现良好

## 示例

**用户**："从Facebook广告库提取Notion的广告，告诉我什么信息传递对他们有效。"

**过程**：
```
正在访问Facebook广告库...
搜索：Notion
找到：23个活跃广告

正在提取截图...
[████████████████████] 100%

保存到：competitor-ads/notion/
- ad-001-collaboration.png
- ad-002-productivity.png
- ad-003-templates.png
...

正在分析信息传递...
```

**输出**：
```
# Notion广告分析

## 概述
- 广告总数：23个活跃
- 主要主题：生产力（35%）、协作（30%）、
  模板（20%）、AI功能（15%）
- 广告格式：静态图片（60%）、视频（40%）
- 行动号召模式："免费试用"、"开始使用"

## 他们强调的关键问题

1. **分散的信息**（8个广告）
   文案："停止在10个不同工具之间切换"
   为什么有效：许多团队面临的直接痛点

2. **会议过载**（5个广告）
   文案："用异步更新替换不必要的会议"
   为什么有效：后COVID远程工作痛点

3. **丢失文档**（4个广告）
   文案："再也不问'那个文档在哪里？'"
   为什么有效：普遍的工作场所挫败感

## 成功的创意模式

### 模式1：前后对比
- 显示混乱的工具环境→ 整洁的Notion工作区
- 用于6个高表现广告
- 视觉隐喻立即清晰

### 模式2：功能展示
- 实际产品使用的GIF
- 在5秒内显示特定功能
- 用于新功能（AI、模板）

### 模式3：社会证明
- "加入2000万用户"信息
- 客户徽标
- 用于4个针对企业的广告

## 有效的文案

最佳标题：
1. "您的团队知识，终于在一个地方"
   → 以痛点为中心的利益导向
   
2. "一体化工作空间"
   → 清晰的定位，广泛的吸引力
   
3. "真正帮助您工作的AI"
   → 解决AI怀疑论，实用角度

最佳正文模式：
- 短句（少于10个词）
- 关注结果而非功能
- 包括具体数字（"减少50%会议"）

## 受众定位洞察

基于广告变化：
- 独角兽创始人：个人生产力角度
- 团队负责人：协作和对齐
- 企业：安全和合规提及
- 学生：免费计划、模板、组织

## 您广告的建议

1. **测试"工具泛滥"痛点**
   → 基于他们的广告频率，强共鸣

2. **使用产品截图而非抽象视觉**
   → 他们所有顶级广告都显示实际UI

3. **先问题后解决方案**
   → "厌倦了X？"比"介绍Y"表现更好

4. **保持文案少于100个字符**
   → 他们最短的广告似乎最频繁

5. **测试前后视觉格式**
   → 他们创意中经过验证的模式

## 保存的文件
- 所有广告：~/competitor-ads/notion/
- 分析：~/competitor-ads/notion/analysis.md
- 最佳表现者：~/competitor-ads/notion/top-10/
```

- 行动号召模式

- 季节性活动
- 产品发布方法
- 功能公告策略

## 最佳实践

### 法律和伦理
✓仅用于研究和灵感
✓不要直接复制广告
✓尊重知识产权
✓使用洞察来告知原创创意
✗不要抄袭文案或窃取设计

### 分析提示
1. **寻找模式**：什么主题重复？
2. **随时间跟踪**：每月保存广告以查看演变
3. **测试假设**：为您品牌调整成功的模式
4. **按受众细分**：不同消息针对不同目标
5. **比较平台**：LinkedIn vs Facebook消息传递不同

## 常见工作流程

### 广告活动规划
1. 提取竞争对手广告
2. 识别成功的模式
3. 注意他们消息中的差距
4. 头脑风暴独特角度
5. 起草测试广告变体

### 定位研究
1. 获取5个竞争对手的广告
2. 映射他们的定位
3. 找到服务不足的角度
4. 开发差异化的信息传递
5. 针对他们的方法进行测试

### 创意灵感
1. 按主题提取广告
2. 分析视觉模式
3. 注意色彩和布局趋势
4. 调整成功的模式
5. 创建原创变体

## 成功的提示

1. **定期监控**：每月检查变化
2. **广泛研究**：也看相邻的竞争对手
3. **保存一切**：构建参考库
4. **测试洞察**：运行您自己的实验
5. **跟踪表现**：A/B测试启发概念
6. **保持原创**：用于灵感，而非抄袭
7. **多个平台**：比较Facebook、LinkedIn、TikTok等

## 输出格式
