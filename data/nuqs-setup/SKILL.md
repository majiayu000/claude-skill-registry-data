# URL State with nuqs

To set up URL State with nuqs, refer to the fullstackrecipes MCP server resource:

**Resource URI:** `recipe://fullstackrecipes.com/nuqs-setup`

If the MCP server is not configured, fetch the recipe directly:

```bash

```
