When create or modify user interfaces, always respect thoses rules :
- Use ux-ui free frameworks and templates
- I want a clean, chic, and elegant look.
- Waiting times in navigation scenarios should always be accompanied by a visual indicator of loading, or a more advanced loading screen if the wait time is long (> 2 seconds). The user experience during these waiting times should be pleasant, attractive, and engaging. 
- Application logo must be used as favicon and icon of web and mobile application.

Keep project technical architecture clear and compliant to thoses rules.
