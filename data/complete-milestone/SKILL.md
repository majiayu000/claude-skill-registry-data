# GSD Complete Milestone

Completes a project milestone.

## When to Use

- Finishing milestone work
- Marking milestone as done

## Process

1. Verify milestone completion
2. Archive milestone documents
3. Update project status

## Success Criteria

Milestone completed and archived.
