# uloop get-unity-info

A tool that returns the current Unity version.

## Usage

```
mcp_uLoopMCP_get-unity-info
```

## Parameters

None

## Response

Returns a JSON object with a `UnityVersion` field containing the Unity version string (e.g., "2022.3.20f1").
