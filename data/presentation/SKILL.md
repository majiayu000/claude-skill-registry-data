# 서비스 기획서 발표자료 작성

## 목적

비즈니스 모델을 기반으로 서비스 기획서 발표자료 스크립트를 작성합니다.

## 사용 시점

- 비즈니스 모델 완성 후 발표 자료 준비가 필요할 때
- gamma.app 등 프레젠테이션 도구로 변환하기 전

## 핵심 요청사항

비즈니스 모델을 기반으로 서비스 기획서 발표자료 스크립트를 작성합니다.

**요청사항**:
- 예시와 같이 각 장표별 요약 작성
- 각 장표의 핵심 메시지는 3개 이하로 요약
- 각 장표 사이를 '---'으로 분리
- Sequential MCP 이용

**참고자료**:
- `think/비즈니스모델.md`

**결과형식**: 마크다운

**결과파일**: `think/서비스기획서스크립트.md`

## 발표자료 표준 구조 (10-15장)
- 비즈니스 모델의 린캔버스 9가지 영역 요약
  - Problem (문제)
  - Customer Segments (고객 세그먼트)
  - Unique Value Proposition (고유 가치 제안)
  - Solution (솔루션)
  - Channels (채널)
  - Revenue Streams (수익 흐름)
  - Cost Structure (비용 구조)
  - Key Metrics (핵심 지표)
  - Unfair Advantage (경쟁우위)

- 비즈니스 모델의 추가분석 요약 
  - 경쟁 분석 및 포지셔닝
  - Go-to-Market 전략
  - 재무 계획

## 스크립트 작성 원칙
- **간결성**: 각 슬라이드당 핵심 메시지 3개 이하
- **스토리텔링**: 문제 → 솔루션 → 가치의 흐름
- **비주얼 중심**: 텍스트 최소화, 비주얼로 설명
- **데이터 기반**: 주장을 데이터로 뒷받침
- **슬라이드 구분**: `---`로 각 슬라이드 구분
