---
name: view-manual
description: |
  Use this skill to view the manual page for a given Tcl command.
---

This system comes with an installation of Tcl 9, the current version of Tcl.

To view the manual page for a Tcl command, you must use `man`:

```bash
man -P cat n list
man -P cat n lappend
man -P cat n expr
```
