# 抽奖获胜者选择器

此技能从列表、电子表格或 Google 表格中为赠品和竞赛随机选择获奖者。

## 何时使用此技能

- 运行社交媒体赠品
- 在活动中抽取抽奖获奖者
- 随机选择调查或测试的参与者
- 从竞赛提交中选择获奖者
- 公平分配有限的名额或资源
- 随机团队分配

## 此技能的作用

1. **随机选择**：使用加密安全的随机选择
2. **多种来源**：支持 CSV、Excel、Google 表格或纯文本列表
3. **多个获奖者**：可以选出一个或多个获奖者
4. **防止重复**：确保同一个人不能赢两次
5. **透明结果**：清晰显示选择过程
6. **获奖者详情**：显示获奖者的所有相关信息

## 如何使用

### 来自 Google 表格

```
从该 Google 表格中随机选择一行以选择赠品的获奖者：[表格链接]
```

### 来自本地文件

```
从 entries.csv 中随机选择 3 个获奖者
```

### 来自列表

```
从该列表中随机选择一个获奖者：
- Alice (alice@email.com)
- Bob (bob@email.com)
- Carol (carol@email.com)
...
```

### 多个获奖者

```
从 contest-entries.xlsx 中随机选择 5 个获奖者，确保不重复
```

## 功能

### 公平选择
- 使用安全的随机数生成
- 无偏见或模式
- 透明过程
- 可重复（带种子，用于验证）

### 排除
```
从该列表中随机选择获奖者，排除之前的获奖者：
Alice、Bob、Carol
```

### 加权选择
```
根据"条目"列以加权概率选择获奖者
（1 个条目 = 1 张票）
```

### 候补获奖者
```
从列表中选择 1 个获奖者和 3 个候补
```

## 示例工作流程

### 社交媒体赠品
1. 将条目从 Google 表单导出到表格
2. "从 [表格链接] 中随机选择获奖者"
3. 验证获奖者详情
4. 公开宣布并附上时间戳

### 活动抽奖
1. 创建包含参会者姓名和邮箱的 CSV
2. "从 attendees.csv 中随机选择 10 个获奖者"
3. 导出获奖者列表
4. 直接邮件通知获奖者

### 团队分配
1. 获得参与者列表
2. "将此列表随机分成 4 个相等的团队"
3. 查看分配
4. 分享团队名单

## 技巧

- **记录过程**：保存时间戳和方法
- **公开宣布**：分享选择详情以保持透明
- **检查资格**：验证获奖者是否符合竞赛规则
- **有备用方案**：选择候补获奖者以防获奖者不符合资格
- **导出结果**：保存获奖者列表作为记录
