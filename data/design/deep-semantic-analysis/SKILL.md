# Deep Semantic Analysis

Use this skill to analyze subtle semantic relationships.

## Workflow

1) Extract key concepts and relations.
2) Compare meanings and context dependencies.
3) Resolve ambiguities with evidence.
4) Summarize aligned vs divergent points.

## Output expectations

- Provide a relation map and ambiguity notes.
