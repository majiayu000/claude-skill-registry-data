# Plugin Enable

無効化されているプラグインを有効化します。

## 引数

- `$ARGUMENTS`: プラグイン名（省略時は一覧から選択）

## オプション

| オプション | 説明                     |
|------------|--------------------------|
| `--help`   | このスキルのヘルプを表示 |

## Claude への指示

**`--help` が指定された場合**: このファイルの内容を要約して表示し、終了。

### 実行手順

1. プラグイン名が指定されていない場合:
   - インストール済みプラグインの一覧を表示
   - 有効化するプラグインの選択を促す

2. プラグインのインストール状態を確認:
   - インストールされていない場合はエラーメッセージを表示して終了
   - 既に有効な場合はその旨を通知して終了

3. 以下のコマンドを実行して有効化:

```bash
claude plugin enable [plugin-name]
```

4. 有効化結果を表示

### 出力フォーマット

```
## プラグインの有効化

プラグイン `shiiman-xxx` を有効化しています...

✅ 有効化完了

プラグイン `shiiman-xxx` が有効になりました。
```

### 重要な注意事項

- ✅ `claude plugin enable` コマンドを使用
- ✅ 有効化前にプラグインのインストール状態を確認
- ✅ 有効化結果をユーザーに報告
- ❌ インストールされていないプラグインを指定された場合はエラーメッセージを表示
- ❌ 既に有効なプラグインを指定された場合はその旨を通知
