# Ops Compliance Runbook

Use this skill to run repeatable audit checks.

## Workflow

1) Select compliance framework and scope.
2) Gather evidence and control mappings.
3) Flag violations and required approvals.
4) Produce an audit summary.

## Output expectations

- Provide control mapping, findings, and remediation steps.
