# Social Media Strategy Skill

## Core Philosophy

> "Social media is not a megaphone. It's a conversation."

In 2024, social media success comes from authentic engagement and platform-native content. Broadcasting doesn't work; building community does. Algorithms favor content that sparks genuine interaction.

## Fundamental Principles

### 1. Platform-Native Content
- Each platform has its own culture and format
- Don't just cross-post the same content everywhere
- Adapt messaging to platform expectations
- Respect platform-specific norms

### 2. Value Over Promotion
- 80/20 rule: 80% value, 20% promotion
- Teach, don't sell
- Share insights, not just links
- Be helpful first, promotional second

### 3. Authentic Voice
- Maintain brand voice while being human
- Engage in two-way conversations
- Respond to comments and mentions
- Be approachable and relatable

### 4. Consistency Over Virality
- Regular posting beats occasional viral hits
- Build trust through reliability
- Sustainable rhythm > burnout cycles

## Platform-Specific Guidelines

### X (Twitter)

**Audience Characteristics:**
- Real-time conversations and breaking news
- 48% of users are there for news updates
- Tech and developer community is very active
- Quick, witty exchanges valued

**Content Strategy:**
```
Best Performing Content Types:
1. Threads (educational, storytelling)
2. Quick tips and insights
3. Hot takes and opinions
4. Behind-the-scenes content
5. Engagement bait (questions, polls)

Avoid:
- External links in main tweets (algorithm penalty)
- Generic promotional content
- Over-hashtagging
- Automated cross-posts
```

```
Single Tweet:
- Max 280 characters, but shorter often performs better
- 1-2 hashtags maximum
- Images increase engagement 150%+
- Keep links in replies, not main tweet

- Hook in first tweet (make it standalone-worthy)
```

### Mastodon

**Audience Characteristics:**
- Tech-savvy, privacy-focused
- Anti-corporate sentiment
- Values authentic interaction
- Decentralized community culture

**Content Strategy:**
```
Best Practices:
- Use content warnings (CW) appropriately
- Alt text on all images (mandatory culture)
- Engage with the community genuinely
- Don't treat it like Twitter/X
- Boost others generously

Avoid:
- Cross-posting from Twitter
- Promotional/marketing tone
- Ignoring CW conventions
- Thread bombing
```

**Format Guidelines:**
```
- 500 character limit (varies by instance)
- Alt text is MANDATORY for images
- Use CW for spoilers, potentially sensitive topics
- Hashtags searchable, use appropriately
- No quote-tweets (by design)
```

### Instagram

**Audience Characteristics:**
- Visual-first platform
- Younger demographic
- Stories for casual, Reels for reach
- Good for data visualization content

**Content Strategy for Data/Tech:**
```
Best Performing Content:
1. Data visualization carousels
2. Infographics
3. Code snippets as images
4. Behind-the-scenes (Reels/Stories)
5. Quick tip videos

Format Specs:
- Feed: 1080x1080 (square) or 1080x1350 (portrait)
- Stories: 1080x1920
- Reels: 1080x1920, 15-90 seconds
```

### YouTube

**Content Strategy for Technical Content:**
```
Video Types:
1. Full tutorials (10-20 min)
2. Quick tips (under 60 seconds for Shorts)
3. Explainers (5-10 min)
4. Live coding sessions
5. Course previews

SEO Elements:
- Keyword in title and description
- Timestamps in description
- Custom thumbnails (1280x720)
- Tags (10-15 relevant)
- Cards and end screens
```

## Engagement Best Practices

### Response Framework

```
When to respond:
- All direct mentions (within 2-4 hours)
- Questions (priority)
- Positive feedback (thank genuinely)
- Constructive criticism (acknowledge)

When NOT to respond:
- Trolls (don't feed them)
- Spam
- Off-topic arguments

Response Tone:
- Friendly but professional
- Helpful first
- Direct when needed
- Grateful for engagement
```

### Building Community

```
Daily Actions:
- Reply to 5-10 relevant conversations
- Share/boost 2-3 pieces of quality content from others
- Engage with followers' content

Weekly Actions:
- Host a Q&A or AMA
- Create a discussion prompt
- Highlight community members

Monthly Actions:
- Review analytics and adjust strategy
- Test new content formats
- Collaborate with others in the space
```

## Metrics to Track

### Engagement Metrics
```
Primary:
- Engagement rate (likes + comments + shares / impressions)
- Reply rate
- Save rate (Instagram)
- Thread read-through rate (Twitter)

Secondary:
- Follower growth rate
- Profile visits
- Link clicks
- Mention sentiment
```

### Benchmarks (Technical/Developer Content)

| Metric | Good | Great | Excellent |
|--------|------|-------|-----------|
| Engagement Rate | 2%+ | 4%+ | 6%+ |
| Reply Rate | 0.5%+ | 1%+ | 2%+ |
| Follower Growth | 2%/month | 5%/month | 10%/month |
