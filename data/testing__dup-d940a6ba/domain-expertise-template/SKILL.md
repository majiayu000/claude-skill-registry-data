# Domain Expertise Template

Copy this template to create your own domain expertise skill.

## Instructions

1. Copy this file to `~/.claude/skills/your-domain/SKILL.md`
2. Replace all `[PLACEHOLDER]` text with your content
3. Customize sections as needed
4. Test and iterate
