# MVP 주제 정의

## 목적

새로운 제품 또는 서비스 아이디어를 위한 명확한 MVP 주제와 비즈니스 도메인을 정의합니다.

## 사용 시점

- 새로운 제품 아이디어 정의가 필요할 때
- 비즈니스 기획 프로세스를 시작할 때
- 비즈니스 도메인 범위를 명확히 할 때
- 사용자가 MVP, 제품 아이디어, 비즈니스 컨셉, 스타트업 아이디어를 언급할 때

## 작성 방법

MVP 주제를 정의할 때 다음 형식으로 분석을 제공합니다:

### 1. MVP 주제
구체적인 비즈니스 도메인을 명확하고 간결하게 설명합니다.

### 2. 선정 이유
MVP후보와 아래 4가지 팩터로 후보들을 평가한 결과를 마크다운 형태로 제공합니다.
- 시장 잠재력: 목표 시장 규모, 성장 트렌드, 현재 시장 공백을 검토
- 사용자 pain points 수준
- 실현 가능성
- 경쟁 우위
  
## 출력 형식

명확한 섹션으로 구조화된 마크다운 출력을 제공합니다. 구체적이고 실행 가능하게 작성합니다.

## 도구 활용

### Sequential MCP 사용
복잡한 도메인 분석이 필요할 때 Sequential MCP를 활용하여 체계적으로 시장 기회를 평가하세요.

## 주의사항
- MVP는 최소한의 핵심 기능에 집중
- 시장 검증이 가능한 범위로 정의
- 3-6개월 내 개발 가능한 규모로 설정
- 명확한 타겟 고객 세그먼트 고려
